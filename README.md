2048++ Strings
==============

## Current Status

Support for the following languages have been published to the Play Store:

* English (en)

Translations for the following languages have been started and are a work in progress:

* German (de)
* Spanish (es)
* French (fr)
* Hindi (hi)
* Italian (it)
* Hebrew (iw)
* Punjabi (pa)
* Portuguese (pt-rBR)
* Romainian (ro)
* Russian (ru)
* Swedish (sv)
* Turkish (tr)
* Ukranian (uk)
* Traditional Chinese (zh-rTW)
